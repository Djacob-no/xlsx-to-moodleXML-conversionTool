This tool creates a webserver that eats specifically formatted excel sheets and spits out moodleXML code to import a whole questionbank to moodle. Supported resources to attach are .pdf .jpg and .png these need to be uploaded to online storage like s3 bucket and a link will be generated to them. The excel sheet needs to be formatted a certain way, see github for example. This tool only supports true/false, multiple choice, multiple select, number and text input questions.
